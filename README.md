# ioVR
VR mod for MICONISOMI's Insult Order (インサルトオーダー)

# 使い方 How to Use It.
Eusth氏の[VRGIN.Template](https://github.com/Eusth/VRGIN.Template)を参照。

インサルトオーダー等のUnity5.6特有のVR化方法については[obtdai/VRGIN](https://github.com/obtdai/VRGIN)を参照。

# VRGIN.Templateからの変更点
- 2D GUIの表示が生成時のVRカメラの傾きを反映してしまうので、補正.
- デフォルトの起動モードを立位に
- IPDScaleのデフォルト値を10に変更
